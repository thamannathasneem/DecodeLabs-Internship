Apple Stock Dashboard - Power BI Project

Project Overview

This project focuses on Data Visualization and Storytelling using Power BI. The dashboard was created using Apple stock market data to analyze stock value trends and present insights visually.

Tools Used

- Power BI
- CSV Dataset
- KPI Cards
- Line Charts

Features

- Interactive dashboard
- KPI summary card
- Stock trend visualization
- Data storytelling approach

Objective

The main objective of this project is to convert raw stock market data into meaningful visual insights that help users understand trends and patterns easily.

Outcome

The dashboard successfully visualizes Apple stock performance over time and demonstrates the use of Power BI for business intelligence and data visualization.
