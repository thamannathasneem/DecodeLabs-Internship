# SQL Data Analysis Project

## Objective
The objective of this project is to perform SQL data analysis using different SQL queries and functions.

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT()
- SUM()
- AVG()

## Platform Used
- SQLite Online

## Dataset Used
Employee dataset containing:
- Employee ID
- Name
- Department
- Salary
- City

## Queries Performed
1. Display all employee records
2. Filter employees using WHERE clause
3. Sort salaries using ORDER BY
4. Group employees using GROUP BY
5. Count total employees
6. Calculate total salary
7. Calculate average salary
8. Perform combined SQL analysis queries

## Conclusion
This project helped in understanding SQL fundamentals, data filtering, sorting, grouping, and aggregation functions used in data analysis.
